# Data License

This example model repository skeleton does not contain real measurement data.

For a real model repository, define the license for:

- pinned datasets;
- submitted result packages;
- generated reports.

Recommended default for original measurement results:

```text
CC-BY-4.0
```

Alternative for maximally reusable public data:

```text
CC0-1.0
```

Do not include third-party datasets or prompts unless their licenses allow redistribution.

The code license of `llmclozestat` does not automatically determine the license of datasets or measurement results.
